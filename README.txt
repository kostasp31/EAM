- Αναφορά εργασίας

Κατά την εκπόνηση της εργασίας υλοποιήθηκαν οι παρακάτω βασικές λειτουργίες της ιστοσελίδας:

	- Δυνατότητα σύνδεσης ή δημιουργίας λογαριασμού (log in/sign up)
	- Αρχική σελίδα (γενική/γονέα/επαγγελματία)
	- Σελίδα βοήθειας (FAQ)

	- Για τους γονείς:
		- Δυνατότητα επεξεργασίας και προβολής προφίλ
		- Δυνατότητα αναζήτησης επαγγελματιών με χρήση φίλτρων
		- Δυνατότητα δημιουργίας ραντεβού και προβολής προηγούμενων ραντεβού
		- Δυνατότητα δημιουργίας αίτησης και προβολής προηγούμενων αιτήσεων
		- Δυνατότητα προβολής τρέχουσας και παλιών συνεργασιών
		- Δυνατότητα πληρωμής με χρήση voucher (χρήση QR code)
		- Δυνατότητα προβολής ειδοποιήσεων
    - Δυνατότητα αξιολόγησης επαγγελματία μετά το πέρας της συνεργασίας

	- Για τους επαγγελματίες:
		- Δυνατότητα επεξεργασίας και προβολής προφίλ
		- Δυνατότητα επεξεργασίας και προβολής βιογραφικού και πιστοποιητικών
		- Δυνατότητα δημιουργίας και επεξεργασίας αγγελίας
		- Δυνατότητα προβολής και αποδοχής ραντεβού
		- Δυνατότητα αποδοχής συμφωνητικού (έναρξη συνεργασίας με γονέα)
		- Δυνατότητα προβολής λεπτομεριών συνεργασίας
		- Δυνατότητα λήψης πληρωμής μέσω του voucher (χρήση QR code)
		- Δυνατότητα προβολής ειδοποιήσεων
    - Δυνατότητα προβολής αξιολογήσεων

Κατα την υλοποίηση του front-end χρησιμοποιήθηκαν οι εξής εκδόσεις του node και npm:
	- node v22.5.1
	- npm 10.8.2

Η ανάπτυξη έγινε σε περιβάλλον Windows 10/11 και σε browsers Chrome και Firefox.
Χρησιμοποιήθηκε Vite Builder για την δημιουργία του project και Firebase για την βάση.
